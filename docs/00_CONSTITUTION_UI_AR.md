# الدستور — واجهة عربية فقط (RTL) [قسم 0bis]

> **المرجع الأعلى:** النسخة الكاملة للدستور في جذر المشروع: **`../.cursorrules`** (ملف Cursor / دستور الهندسة).  
> هذا الملف يعيد نشر **§0bis** كي يكون مرئياً داخل `docs/` وللطباعة أو تضمينه في عقود التنفيذ.

## 0bis) اللغة والواجهة (إلزامي)

1. **واجهة المستخدم: عربية فقط (RTL).**  
   لا قوائم تنقل ولا عناوين شاشات بالإنجليزية في **بيئة الإنتاج** للمستخدم النهائي.

2. **استثناءات مسموحة (واضحة ومحدودة):**
   - رموز العملات الدولية المعيارية، مثل: `ILS`, `USD`, `EUR`, `JOD`.
   - الأرقام والمبالغ داخل خلايا جداول أو حقول إدخال رقمي يمكن عرضها بـ **`dir="ltr"`** لمحاذاة صحيحة.
   - أسماء تقنية **داخل الكود** (متغيرات، ملفات، مسارات API) بالإنجليزي — **ولا تُعرض** للمستخدم كما هي إلا إذا كان ذلك ضرورياً تقنياً ومُفسَّخاً في واجهة عربية (مثلاً معرف فني في تفاصيل دعم فقط).

3. **الهوية البصرية:**  
   اتبع **`docs/05_VISUAL_IDENTITY_AR.md`** وملف الشعار **`branding/logo-official.png`**.

- أي شاشة أو تقرير أو تصدير يظهر للمستخدم **يجب** أن يمر بمراجعة: **عربي + RTL + استثناءات أعلاه فقط**.
