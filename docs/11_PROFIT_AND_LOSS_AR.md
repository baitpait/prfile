# قائمة الربح والخسارة — بروفايل ميدا

## التقارير الثلاثة

| المسار | الوصف |
|--------|--------|
| `/reports/profit-loss` | **كامل (فواتير)** — استحقاق |
| `/reports/profit-loss-cash` | **بدون دين (نقدي)** — دفعات فقط |
| `/reports/profit-loss-ils` | **بالشيكل** — تحويل BOI |

## المعادلة (لكل عملة منفصلة)

**كامل:**
```
صافي = مبيعات (فواتير issued) − مشتريات (أوامر issued) − مصروفات − رواتب مدفوعة
```

**بدون دين:**
```
صافي = دفعات عملاء − دفعات موردين − مصروفات − رواتب مدفوعة
```

- **لا COGS** — لا تكلفة مبيعات من الكتالوج.
- **الرواتب** من `salary_payments` فقط (`status=paid`, `paid_at` في الفترة).
- **لا دمج عملات** في رقم واحد إلا تقرير الشيكل.

## أسعار الصرف

- مصدر: **بنك إسرائيل** (שער יציג)
- جدول: `exchange_rates`
- أمر يومي: `php artisan boi:fetch-rates` (مجدول 16:30)
- التقرير بالشيكل يستخدم سعر تاريخ **نهاية الفترة**
