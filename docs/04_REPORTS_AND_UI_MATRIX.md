# مصفوفة التقارير والواجهات — بروفايل ميدا

للربط مع **المسارات الفعلية** و**مكوّنات Livewire** و**تجميعات لوحة التحكم حسب العملة** راجع `docs/07_SYSTEM_OVERVIEW_AR.md`.

## مبادئ الواجهة
- **لوحة تنفيذية:** بطاقات (إيرادات الشهر، فواتير معلّقة، مشاريع متأخرة، دفعات مستحقة).
- **قوائم قابلة للتصفية:** عميل، مورد، عملة، تاريخ، حالة.
- **تفاصيل كيان:** تبويبات: نظرة عامة | مستندات | دفعات | جهات اتصال | سجل نشاط (حسب التوفّر).
- **طباعة/PDF:** قالب تقرير رسمي (شعار، تذييل، ترقيم صفحات).

## التقارير حسب العملة
| التقرير | الجمهور | المخرجات |
|---------|---------|-----------|
| كشف حساب عميل | محاسب/مدير | أقسام ILS / USD / … مع مدين/دائن/رصيد لكل عملة |
| كشف حساب مورد | محاسب | نفس المنطق |
| أعمار الذمم | إدارة | فواتير مفتوحة حسب الأيام + العملة |
| ملخص مبيعات فترة | إدارة | حسب العملة؛ بدون جمع عبري غير موثّق |
| مصروفات فترة | محاسب | حسب الوصف النصي + التصنيف اليدوي لاحقاً إن رغبتم |
| مشاريع حسب الحالة | تنفيذ | Kanban أو جدول + SLA |

## الصلاحيات على التقارير
- **قراءة فقط:** تقارير مجمّعة بدون تعديل.
- **محاسب:** دفعات، فواتير، كشوف.
- **مدير:** كل شيء + إعدادات.

## مخرجات التصدير
- CSV/Excel للجداول الطويلة.
- PDF للكشوف الرسمية.

## لوحات مستقبلية (Phase 2+)
- توقع تدفق نقدي بسيط (اختياري).
- تكامل بريد/تذكيرات (ADR منفصل).
